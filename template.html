카카오맵 키 설정 (최초 1회)

지도를 그리려면 무료 카카오 JavaScript 키가 필요해요.

  1. developers.kakao.com 로그인 → 내 애플리케이션 추가
  2. 앱 키 → JavaScript 키 복사
  3. 앱 설정 → 플랫폼 → Web 에 사용할 주소 등록
    (로컬 실행이면 http://localhost:8000 을 등록하고 아래 README 방식으로 여세요)

키는 이 브라우저에만 저장됩니다(localStorage).
