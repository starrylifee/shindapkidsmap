# 우리 동네 추천 장소 지도 🗺️

신답초 학부모회 Padlet(우리 동네 추천 장소)을 분석·정리해 **지도 시각화**로 만든 자료입니다.
마커를 클릭하면 장소명·주소·추천 이유·사진이 보입니다. 학년별 색상 구분과 검색 기능 포함.

## 📁 파일 구성
| 파일 | 설명 |
|------|------|
| `index.html` | **메인 지도** (브라우저에서 열기) |
| `정리된_추천장소.csv` | 정리된 표 데이터 (엑셀에서 열기 가능, 172곳) |
| `data.json` | 지도가 사용하는 원본 데이터 |
| `build_data.py` / `gen_html.py` / `template.html` | 데이터 추출·지도 생성 스크립트 (재생성용) |

## 🚀 지도 여는 방법

지도는 **카카오맵**을 쓰기 때문에 무료 키 발급 + 로컬 서버 실행이 필요합니다.

### 1단계 — 카카오 JavaScript 키 발급 (최초 1회, 약 5분)
1. https://developers.kakao.com 로그인
2. **내 애플리케이션 → 애플리케이션 추가** (이름 아무거나)
3. **앱 키 → JavaScript 키** 복사
4. 같은 앱의 **플랫폼 → Web 플랫폼 등록** 에 `http://localhost:8000` 입력 후 저장

### 2단계 — 로컬 서버로 열기
이 폴더에서 `start.bat` 더블클릭하면 자동으로 서버가 켜지고 브라우저가 열립니다.
(수동: 폴더에서 `python -m http.server 8000` 실행 후 브라우저에서 `http://localhost:8000` 접속)

### 3단계 — 키 입력
처음 열면 키 입력창이 뜹니다. 1단계에서 복사한 **JavaScript 키**를 붙여넣으면 끝.
(키는 브라우저에만 저장되며, 다음부터는 묻지 않습니다.)

## ✨ 기능
- 학년별(1~6학년) 색상 마커 + 필터 칩
- 왼쪽 목록 클릭 → 해당 마커로 이동 + 상세 패널
- 마커 클릭 → 사진·주소·추천 이유 표시
- 장소명/이유 키워드 검색
- 위치 좌표는 한 번 찾으면 캐시되어 다음 접속 시 빠르게 표시

## 📊 데이터 요약
- 총 **172곳** (1학년 35 · 2학년 24 · 3학년 36 · 4학년 52 · 5학년 15 · 6학년 10)

- 가장 많이 추천된 곳: 간데메공원(7회), 배봉산숲속도서관·답십리도서관·청량꿈숲 등(3회)

> 일부 게시물은 주소가 없어 **장소명으로 검색**해 위치를 찾습니다.
> 위치가 어긋난 곳은 `정리된_추천장소.csv`에서 주소를 보완 후 `python build_data.py && python gen_html.py` 로 다시 생성하세요.
